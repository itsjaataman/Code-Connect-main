## 🚀 Getting Started

### 1. Clone the Repository
```bash
git clone https://github.com/itsjaataman/Code-Connect-main.git
cd Code-Connect-main
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Run the Project
```bash
npm start
```
